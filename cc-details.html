Anime Face Generator

Project information

  • Project URL: Github.com

During my Master's degree in Computer Games at Queen Mary University of London, I developed a project focused on generating anime faces using Generative Adversarial Networks (GANs). GANs are a class of artificial intelligence algorithms used in unsupervised machine learning, where two neural networks contest with each other in a zero-sum game framework. In this project, I trained a GAN model on a dataset of anime faces to learn the underlying patterns and features of anime-style artwork. The trained model was then able to generate new and unique anime faces that closely resembled the style of the input dataset. I implemented the project using TensorFlow and Python, leveraging the power of deep learning techniques to create visually appealing anime faces. The project involved data preprocessing, model training, hyperparameter tuning, and evaluation of generated results. By completing this project, I gained valuable experience in deep learning, computer vision, and generative modeling techniques. I also developed skills in data manipulation, model optimization, and project management, further enhancing my expertise in artificial intelligence and machine learning.
